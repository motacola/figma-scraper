## Chrome Web Store Requirements

### Image Specifications:
- **Format**: PNG or JPEG
- **Recommended size**: 1280x800 or 640x400
- **Maximum**: 5 screenshots
- **File size**: Each image must be under 5MB

### Current Screenshots:
All screenshots are approximately 350px wide (extension popup size) and will need to be:
1. Resized to meet store requirements (1280x800 recommended)
2. Optionally placed on a background showing Chrome browser context
3. Annotated with callouts highlighting key features (optional)

## Usage

Upload these screenshots to the Chrome Web Store Developer Dashboard in the "Store listing" section.

### Recommended Order:
1. Main Interface (shows primary functionality)
2. Flow Discovery (shows automation feature)
3. Capture Progress (shows it working)
4. License Screen (shows entry point)
